Expand description

Helper trait for Renderer, which defines a rendering api for a currently in-progress frame during Renderer::render.

fn clear(
    &mut self,
    color: [f32; 4],
    at: &[Rectangle<i32, Physical>]
) -> Result<(), Self::Error>

Clear the complete current target with a single given color.

The at parameter specifies a set of rectangles to clear in the current target. This allows partially clearing the target which may be useful for damaged rendering.

This operation is only valid in between a begin and finish-call. If called outside this operation may error-out, do nothing or modify future rendering results in any way.

Render part of a texture as given by src to the current target into the rectangle described by dst as a flat 2d-plane after applying the inverse of the given transformation. (Meaning src_transform should match the orientation of surface being rendered).
