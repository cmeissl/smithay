Module dbus::arg::messageitem

source ·
Expand description

MessageItem - old, enum design that is used as parameters and return values from method calls, or as data added to a signal.

Note that the newer generic design (see arg module) is, in general, both faster and smaller than MessageItem, and should be your first hand choice whenever applicable. There is also a trait object design called RefArg in case the generic design is too inflexible.
